## 14.1 AI绘画工作流

### 14.1.1 场景描述

**用户**：内内容创作者、设计师、自媒体运营、教程作者
**需求**：快速生成配图、白板图、Logo设计、批量处理
**工具**：Gemini图像生成、官方 `image_generate`、ComfyUI、Midjourney、Stable Diffusion

**痛点**：
❌ 手机无法使用：专业绘图工具只有网页版
❌ 切换麻烦：需要在多个工具间切换
❌ 修改困难：每次修改都要重新上传
❌ 批量处理慢：一张一张处理效率低
❌ 成本高昂：专业工具订阅费贵


**目标**：
✅ 手机随时画图
✅ 统一入口操作
✅ 语音修改图片
✅ 批量自动处理
✅ 成本可控

**命令行批量生成：**

如果你需要为整本教程生成配图，可以创建批量脚本：

```bash
#!/bin/bash
# batch_generate_chapters.sh

chapters=(
    "第1章节:OpenClaw是什么"
    "第2章节:部署方式对比"
    "第3章节:人设配置"
    "第4章节:文件管理"
    "第5章节:知识库管理"
    # ... 更多章节节
)

for chapter in "${chapters[@]}"; do
    IFS=':' read -r num title <<< "$chapter"
    
    echo "生成 ${num} - ${title}"
    
    openclaw infer image generate \
        --prompt "生成一张白板图片，手写字体风格，总结${title}的核心要点..." \
        --api-format openai \
        --json
    
    echo "✅ ${num} 完成"
    sleep 2  # 避免API限流
done

echo "🎉 所有章节节配图生成完成！"
```
